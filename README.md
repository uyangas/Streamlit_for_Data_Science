# Streamlit-г Data Science-д ашиглан интерактив апп бүтээх нь

## Шаардлага:
`requirements.txt` файлыг ашиглан шаардлагатай сангуудыг суулгах
- `python 3.8`
- `streamlit`
- `bokeh==2.4.3`
- `pydeck`
- `altair`
- `matplotlib`
- `seaborn`
- `openpyxl`
- `scikit-learn`
- `nbformat` - хэрэв `ipynb` ашиглаж байгаа тохиолдолд шаардаж байгаа бол суулгана
